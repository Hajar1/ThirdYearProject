SYSC3110 Team SmallTalkers (Network Routing Simulator) - Milestone #1

System Description:

A simulator to evaluate the performance of various routing algorithms


Setup Instructions:

1.Import the project into Eclipse by going to import-> Existing Projects into Workspace
2.Select the folder SYSC3110_Milestone_1
3.Run NetworkUI first by right clicking and selecting "Run as Java Application"
4.You will see the results displayed on the console
